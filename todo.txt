
ASCII

На самом деле тут не совсем ascii, а можно задавать служебные символы или символы с каким-то кодом.

Примером служебного символа может быть прозрачный символ, который не заменяет тот на который выводится.

Поэтому тут будет типа

enum Char {
    Transparent
    Printable { ... }
}

То же самое цвета:

enum Color {
    Transparent
    Indexed (u8)
    Rgba { ... }
}
